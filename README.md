# Multiplayer Third Person Shooter


Godot 3.2
=========

**A multiplayer third person shooter example project for Godot 3.2 game engine.**

Features
========

- Looking around
- Walking/sprinting
- Jumping
- Sounds
- A map
- Weapon pickups
- Vehicles

Keys
====

- Walk: `W/A/S/D`
- Sprint: `Left Shift`
- Jump: `Space`
- Fire: `Left Mouse Button`
- Reload: `R`
